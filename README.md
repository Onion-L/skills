# onion/skills

A curated collection of Claude Code skills that supercharge your landing page workflow — from exploration to production in minutes.

## Skills

| Skill | What it does |
|---|---|
| `design-gallery` | Generate 5 distinct landing page variations from your project or PRD |
| `redesign` | Not happy with a variation? Redesign it without repeating directions |
| `finalize` | Promote your chosen variation to a production-ready page with real copy and SEO |

## Workflow

```
/design-gallery  →  pick a variation  →  /redesign (optional)  →  /finalize
```

## Installation

```bash
npx skills add {your-github-username}/{repo-name}
```
